Comment utiliser ce script ?

Il est nécéssaire d'importer dans son code le script inclut dans le fichier maps.js

Ensuite, il faut configurer ce script dans son code. Il faut simplement rajouter la ligne suivante :

<div class="google-map" data-marker="TRUE/FALSE" data-marker-icon="MARKER FILE" data-center-lat="XX.XXXX" data-center-lng="XX.XXXX" data-zoom="XX"></div>

Vous trouverez plusieurs points de configuration :

  • bool data-marker : Permet de modifier le marqueur Google Maps par défaut option : true / false.
  • string data-marker-icon : Si data-marker="true", indiquer le chemin vers le fichier image du marqueur.
  • float data-center-lat : Latitude de l'emplacement à afficher.
  • float data-center-lng : Longitude de l'emplacement à afficher.
  • float data-zoom : valeur du zoom sur la carte.
Une fois cette configuration effectué, il faut définir une hauteur (height) afin d'afficher la carte. Dans cet exemple, la hauteur est défini dans la classe css .google-map.
